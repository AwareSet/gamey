Interactive Group Quiz

Main Features

Game Creation

  • New Game: Allows users to create a new game.
  • My Games: Lets users restore or continue previous games.
  • Children's Interface: Offers a child-friendly version of the game.

Category Selection

Players select categories for their team and the opposing team. Each team selects 3 categories, making a total of 6 categories with 36 different questions.

Team Information

Players specify the game name and team names, along with the number of players in each team.

Detailed Steps

1. Create a Game

When players first enter the game interface, they are greeted with options to create a new game, access their previous games, or switch to a children's interface.

  • New Game: Initiates the process of setting up a new quiz game.
  • My Games: Allows players to continue or review games they have previously started.
  • Children's Interface: Provides a simplified version of the game suitable for younger players.

2. Select Categories

Players are then prompted to select categories for their quiz. The categories are divided into two sections:

3. Specify Team Information

After selecting the categories, players need to provide information about the teams:

  • Game Name: Enter a name for the game to easily identify it.
  • First Team: Enter the name of the first team and specify the number of players.
  • Second Team: Enter the name of the second team and specify the number of players.

4. Start the Game

Once the categories and team information are set, players can start the game by clicking the Start playing button. This button is prominently displayed at the bottom of the interface, ensuring easy access.

Summary

1. Create a Game

Players initiate a new game or continue a previous one.

2. Select Categories

Players choose categories for both teams from a variety of topics.

3. Specify Team Information

Players enter the game name, team names, and the number of players.

4. Start the Game

Players begin the quiz by clicking the "Start playing" button.
